# Hi, I build beginner-friendly machine learning projects

I’m working on a Titanic survival prediction project that shows the full ML workflow:
exploration, preprocessing, training, evaluation, and experiment tracking.

## Featured Project

### Titanic Survival Prediction
A small but complete ML project using the Titanic dataset.

**What it includes:**
- data cleaning and preprocessing
- feature engineering with `FamilySize`, `IsAlone`, and `Title`
- model comparison with Logistic Regression, Random Forest, and Gradient Boosting
- evaluation with accuracy, precision, recall, F1, and confusion matrices
- experiment logs and interactive notebook summaries

**Best result so far:**
- engineered features improved Logistic Regression to the best test F1 score in the current workflow

**Repo link:**
- https://github.com/Neupane-pradip/-Titanic-Survival-Prediction.git
